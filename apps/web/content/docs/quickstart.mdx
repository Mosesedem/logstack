# Quick Start

Get Logstack integrated into your application in 5 minutes.

<Steps>

<Step>
## Create an Account

Sign up for Logstack at [logstack.tech/signup](https://logstack.tech/signup) or deploy your own instance.

</Step>

<Step>
## Create a Project

After signing in, create a new project from the dashboard. You'll receive an API key starting with `ls_live_`.

<Callout type="warn">
  Keep your API key secure. Never commit it to version control or expose it in
  client-side code.
</Callout>

</Step>

<Step>
## Install the SDK

<Tabs items={["JavaScript", "Go", "Python"]}>
  <Tab value="JavaScript">

```bash
npm install logstack-js
# or: pnpm add logstack-js / yarn add logstack-js
```

  </Tab>
  <Tab value="Go">

```bash
go get github.com/mosesedem/logstack/packages/logstack-go-sdk@v1.0.3
```

  </Tab>
  <Tab value="Python">

```bash
pip install logstack-py
```

  </Tab>
</Tabs>

</Step>

<Step>
## Initialize and log

<Tabs items={["JavaScript", "Go", "Python"]}>
  <Tab value="JavaScript">

```typescript title="src/lib/logger.ts"
import { createLogStack } from "logstack-js";

export const logger = createLogStack({
  apiKey: process.env.LOGSTACK_API_KEY!,
  // captureConsole defaults to true — existing console.* calls are shipped
});
```

```typescript title="src/index.ts"
import { logger } from "./lib/logger";

logger.info("Application started", { version: "1.0.0" });
logger.info("User action", {
  userId: "user_123",
  action: "login",
});

try {
  throw new Error("Something went wrong");
} catch (error) {
  logger.error("Operation failed", {
    error: (error as Error).message,
    stack: (error as Error).stack,
  });
}
```

  </Tab>
  <Tab value="Go">

```go title="main.go"
package main

import (
	"context"
	"os"

	logstack "github.com/mosesedem/logstack/packages/logstack-go-sdk"
)

func main() {
	client := logstack.NewClient(logstack.Config{
		APIKey:      os.Getenv("LOGSTACK_API_KEY"),
		Environment: "production",
	})
	defer client.Close()

	ctx := context.Background()
	_ = client.Info(ctx, "Application started", map[string]interface{}{
		"version": "1.0.0",
	})
	_ = client.Info(ctx, "User action", map[string]interface{}{
		"userId": "user_123",
		"action": "login",
	})
}
```

  </Tab>
  <Tab value="Python">

```python title="main.py"
import os
from logstack import LogStackClient

with LogStackClient(
    api_key=os.environ["LOGSTACK_API_KEY"],
    environment="production",
) as client:
    client.info("Application started", metadata={"version": "1.0.0"})
    client.info(
        "User action",
        metadata={"userId": "user_123", "action": "login"},
    )
```

  </Tab>
</Tabs>

</Step>

## Environment variables

```bash title=".env"
# Required: project ingest key from the dashboard
LOGSTACK_API_KEY=ls_live_your_api_key_here

# Optional: self-hosted API host (no /v1 suffix)
LOGSTACK_ENDPOINT=https://api.logstack.tech
# Go SDK reads the same idea as APIURL; Python as api_url
```

<Callout type="info">
  Add <code>.env</code> to <code>.gitignore</code> so secrets never land in git.
</Callout>
