# Welcome to Logstack

Logstack is a **production-ready log management platform** for modern applications.
Capture, stream, and analyze logs in real time with alerting, a web dashboard, and a
mobile companion app.

<Callout type="info">
  Open source and self-hostable. Get started in minutes with a single API key.
</Callout>

## Key features

- **Real-time streaming** — dashboards and mobile as events land
- **Smart alerts** — pattern rules with email and push
- **Multi-language SDKs** — JavaScript/TypeScript, Go, and Python
- **Self-hosted** — full control of your data

## Install an SDK

<Tabs items={["JavaScript", "Go", "Python"]}>
  <Tab value="JavaScript">

```bash
npm install logstack-js
```

```typescript
import { createLogStack } from "logstack-js";

const logstack = createLogStack({
  apiKey: "ls_live_your_api_key",
});

logstack.info("User signed up", { userId: "user_123" });
```

  </Tab>
  <Tab value="Go">

```bash
go get github.com/mosesedem/logstack/packages/logstack-go-sdk@v1.0.3
```

```go
client := logstack.NewClient(logstack.Config{
	APIKey: "ls_live_your_api_key",
})
defer client.Close()

_ = client.Info(ctx, "User signed up", map[string]interface{}{
	"userId": "user_123",
})
```

  </Tab>
  <Tab value="Python">

```bash
pip install logstack-py
```

```python
from logstack import LogStackClient

with LogStackClient(api_key="ls_live_your_api_key") as client:
    client.info("User signed up", metadata={"userId": "user_123"})
```

  </Tab>
</Tabs>
