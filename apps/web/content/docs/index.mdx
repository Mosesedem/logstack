# Welcome to Logstack

Logstack is a **production-ready log management platform** designed for modern applications. Capture, stream, and analyze logs in real-time with powerful alerting and beautiful dashboards.

<Callout type="info">
  Open source and self-hostable. Get started in minutes with a single API key.
</Callout>

## Key Features

- **Real-time Streaming** — See logs as they happen across your dashboards and mobile app
- **Smart Alerts** — Automatic pattern detection with smart notifications via email or push
- **Multi-platform** — Use our SDKs, web dashboard, or mobile apps
- **Self-hosted** — Full control of your data on your infrastructure

## Getting Started

<Tabs items={["npm", "yarn", "pnpm"]}>
  <Tab value="npm">```bash npm install logstack-js ```</Tab>
  <Tab value="yarn">```bash yarn add logstack-js ```</Tab>
  <Tab value="pnpm">```bash pnpm add logstack-js ```</Tab>
</Tabs>

```typescript title="Send your first log"
import { createLogStack } from "logstack-js";

const logstack = createLogStack({
  apiKey: "your_api_key_here",
});

logstack.info("User signed up", { userId: "user_123" });
```
